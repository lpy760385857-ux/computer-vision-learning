# 用于句子分类的卷积神经网络

论文《用于句子分类的卷积神经网络》(EMNLP 2014) 的代码实现。

在 Pang 和 Lee 的电影评论数据集（论文中的 MR 数据集）上运行模型。

使用数据时请引用原论文。

## 环境要求

代码使用 Python (2.7) 编写，需要 Theano (0.7)。

若使用预训练的 word2vec 向量，还需要从以下链接下载二进制文件：

[https://code.google.com/p/word2vec/](https://code.google.com/p/word2vec/)

## 数据预处理

要处理原始数据，请运行：

text

```
python process_data.py path
```

其中 path 指向 word2vec 二进制文件（即 GoogleNews-vectors-negative300.bin 文件）。

这会在相同文件夹下生成一个名为 mr.p 的 pickle 对象，其中包含格式正确的数据集。

注意：这将创建与论文中使用不同折叠分配的数据集。

不过，你仍然可以通过 CNN-nonstatic 模型获得 >81% 的交叉验证分数。

## 运行模型 (CPU)

示例命令：

text

```
THEANO_FLAGS=mode=FAST_RUN,device=cpu,floatX=float32 python conv_net_sentence.py -nonstatic -rand
THEANO_FLAGS=mode=FAST_RUN,device=cpu,floatX=float32 python conv_net_sentence.py -static -word2vec
THEANO_FLAGS=mode=FAST_RUN,device=cpu,floatX=float32 python conv_net_sentence.py -nonstatic -word2vec
```

这三条命令分别对应运行论文中的 CNN-rand、CNN-static 和 CNN-nonstatic 模型。

## 使用 GPU

GPU 可带来 10 到 20 倍的良好加速，因此强烈推荐使用。

要使用 GPU，只需将 device=cpu 改为 device=gpu（或你所使用的具体 gpu）。

例如：

text

```
THEANO_FLAGS=mode=FAST_RUN,device=gpu,floatX=float32 python conv_net_sentence.py -nonstatic -word2vec
```
