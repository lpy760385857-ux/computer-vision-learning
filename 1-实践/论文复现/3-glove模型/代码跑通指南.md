# glove-python

这是一个 [GloVe](http://www-nlp.stanford.edu/projects/glove/) 的简易 Python 实现版本。

GloVe 用于生成单词的稠密向量嵌入 (dense vector embeddings)，在最终的向量空间中，经常一起出现的单词彼此距离会比较近。

虽然这生成的嵌入与 [word2vec](https://code.google.com/p/word2vec/)（其在 [gensim](http://radimrehurek.com/gensim/models/word2vec.html) 中有一个非常棒的 Python 实现）类似，但它们的方法不同：GloVe 是通过对语料库单词共现矩阵的对数进行分解来生成嵌入的。

该代码使用了异步随机梯度下降 (asynchronous stochastic gradient descent) 算法，并使用 Cython 实现。很有可能，里面包含了大量的 bug。

## 安装 (Installation)

使用 pip 从 pypi 安装：`pip install glove_python`。

**OSX 用户注意：**由于使用了 OpenMP，`glove-python` 无法在 Clang 下编译。要安装它，您需要一个版本相对较新的 `gcc`（例如通过 Homebrew 安装）。`setup.py` 应该会自动识别它；如果没有识别到，请提交一个 issue。

也不支持使用 OSX 自带的默认 Python 发行版进行构建；请尝试使用 Homebrew 或 Anaconda 提供的版本。

## 用法 (Usage)

生成嵌入是一个两步过程：首先从语料库创建共现矩阵，然后使用该矩阵生成嵌入。`Corpus` 类可以帮助您从可迭代的标记 (tokens) 中构建语料库；`Glove` 类用于训练嵌入（提供类似于 sklearn 的 API）。

此外，它还支持基础的段落向量 (paragraph vectors)。段落向量（在此处）是指将段落（包含多词的文本片段）嵌入到词向量空间中，使得段落的表示与它所包含的单词相近，同时根据语料库中的单词频率进行了调整（类似于 tf-idf 权重方式）。在训练完词向量后，可以通过在训练好的模型上调用 `transform_paragraph` 方法来获取这些段落向量。

## 示例 (Examples)

`example.py` 包含了一些运行简单训练脚本的示例代码：运行 `ipython -i -- examples/example.py -c my_corpus.txt -t 10` 将处理您的语料库，运行 10 个 GloVe 训练周期 (epochs)，然后让您进入一个 `ipython` 交互界面。在那里输入 `glove.most_similar('physics')` 应该会生成一个相似词汇列表。

如果您想处理维基百科语料库，可以使用 `-w` 标志将[此处](http://dumps.wikimedia.org/enwiki/latest/)的文件传入 `example.py` 脚本。运行 `make all-wiki` 会下载一个较小的维基百科数据转储文件，对其进行处理，并训练嵌入。构建共现矩阵需要花费一些时间；您可以通过增加训练并行度以匹配可用的物理 CPU 核心数量来加快向量训练的速度。
